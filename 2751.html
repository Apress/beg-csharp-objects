Chapter 3, page 80, 2nd paragraph:

 

The phrase they use Pascal casing should read they use camel casing

 

Names for (non-attribute) reference variables follow the same convention as method and attribute names: i.e., they use camel casing. Some sample reference variable declarations are as follows:

  Student x;

  Student aStudent;

  Course prerequisiteOfThisCourse;

  Professor myAdvisor;
